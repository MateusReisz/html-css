Tags em HTML

A seguir algumas tags muito importantes para a criação de sites:

  • Criar parágrafos;
  • Criar titulos/subtitulos;
  • Adicionar favicon ao lado do titulo do cabeçalho;
  • Colocar trechos em formato de lista;
  • Listar palavras/itens;
  • Marcar texto de forma simples;
  • Marcar todos os textos com a mesma cor;
  • Adicionar endereço;
  • Usar frases em "itálico";
  • Para adicionar letras pequenas (letras miudas em doc);
  • Indicar texto excluído;
  • Indicar texto inserido;
  • Texto sobrescrito (Utilizado em potenciação);
  • Texto Subescrito (Utilizado em fórmula quimíca,ex: H2O);
  • Código-fonte e Pré-formatação;
  • Citações simples;
  • Citações completas + link;
  • Abreviações;
  • Adicionar imagens;
  • Adicionar emojis;
  • Fazer quebra de linha;
  • Usar os sinais de < ou >;
